## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| `‚åòK` / `Ctrl+K` | Open Command Palette ‚Äî search tasks, projects, clients, invoices, and navigate anywhere |
| `‚åò\` / `Ctrl+\` | Toggle sidebar collapse |
| `‚åò‚áßP` | Create new project |
| `?` | Open keyboard shortcuts help overlay |

### Command Palette

The Command Palette (`‚åòK`) provides universal search across all your data and instant navigation to any page.

| Shortcut | Action |
|----------|--------|
| `‚Üë` / `‚Üì` | Navigate results |
| `Enter` | Select highlighted result |
| `Escape` | Close palette |

<Callout kind="info">
Navigation commands are **role-filtered** ‚Äî client users only see pages they can access. When focused inside the rich text editor, `‚åòK` opens the insert link dialog instead.
</Callout>

### Go To Navigation

Press `G`, then a second key within 800ms to jump directly to a page:

| Sequence | Destination |
|----------|-------------|
| `G` ‚Üí `D` | Dashboard |
| `G` ‚Üí `T` | Tasks |
| `G` ‚Üí `P` | Projects |
| `G` ‚Üí `M` | Messages |
| `G` ‚Üí `I` | Invoices |
| `G` ‚Üí `S` | Settings |

## Tasks Page

Shortcuts available on the Tasks page. These are ignored when focus is inside a text input.

| Shortcut | Action |
|----------|--------|
| `/` | Focus the task search input |
| `N` or `C` | Open new task creation |
| `‚åòEnter` | Submit new task (when creation prompt is open) |
| `1` | Switch to List view |
| `2` | Switch to Board view |
| `3` | Switch to Workload view |

### List View Navigation

Navigate and interact with tasks using your keyboard in the List view:

| Shortcut | Action |
|----------|--------|
| `J` / `‚Üì` | Move focus to next task |
| `K` / `‚Üë` | Move focus to previous task |
| `Enter` | Open focused task in drawer |
| `X` | Toggle complete on focused task |
| `E` | Inline edit focused task title |

<Callout kind="tip">
Focused rows show a highlighted accent border and auto-scroll into view. When inline editing (`E`), press `Enter` to save or `Escape` to cancel.
</Callout>

## Task Drawer

The Task Drawer supports **Tab-prefix shortcuts** ‚Äî press `Tab`, then a follow-up key within 600ms. These are ignored when focus is inside a text input or editor.

| Shortcut | Action |
|----------|--------|
| `Escape` | Close the drawer |
| `Tab` ‚Üí `S` | Switch to Subtasks tab and focus add-subtask input |
| `Tab` ‚Üí `A` | Open the assignee picker |
| `Tab` ‚Üí `D` | Open the due date picker |
| `Tab` ‚Üí `P` | Cycle priority (Low ‚Üí Medium ‚Üí High ‚Üí Urgent) |
| `Tab` ‚Üí `C` | Switch to Details tab and focus comment editor |
| `Tab` ‚Üí `M` | Assign task to yourself |
| `Tab` ‚Üí `F` | Toggle fullscreen drawer |
| `‚åòD` / `Ctrl+D` | Duplicate task |
| `Tab` ‚Üí `Delete` | Delete task (with confirmation) |

## Rich Text Editor

Available in task descriptions, comments, and any rich text field.

### Text Formatting

| Shortcut | Action |
|----------|--------|
| `‚åòB` / `Ctrl+B` | Toggle **bold** |
| `‚åòI` / `Ctrl+I` | Toggle *italic* |
| `‚åòU` / `Ctrl+U` | Toggle underline |
| `‚åòK` / `Ctrl+K` | Insert or edit link |
| `Tab` | Indent list item |
| `Shift+Tab` | Outdent list item |

### @Mention Autocomplete

| Shortcut | Action |
|----------|--------|
| `‚Üë` / `‚Üì` | Navigate mention suggestions |
| `Enter` / `Tab` | Accept selected mention |
| `Escape` | Dismiss mention dropdown |
